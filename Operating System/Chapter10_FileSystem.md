# 5. File Sharing(파일 공유)
File sharing이란 네트워크를 이용하여 시스템들 간에 파일 시스템을 접근 하는 방법이다. 

기본 구조는 Clinet-Server를 통해 서버로 부터 파일 시스템을 마운팅 하여 클라이언트가 접근 하도록 한다.

이때 UNIX는 NFS를, Windows는 CIFS 형식을 사용한다.
